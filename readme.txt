# Gul Awaiz - Web Developer

## Introduction

This website project is to showcase the ability to create a website using only HTML, Css and Javascript/ jQuery. This website explain who I am and what I can do. 

It is responsive website and uses bootstrap styles and script to make it responsive. The menu of the website is also responsive you can resize the screen to check the menu behaviour changing when the screen size is changed. 

It contains four html pages named, index.html, resume.html, portfolio.html and contact.html. The stylesheet and script are in separate files named as style.css and js.js.
